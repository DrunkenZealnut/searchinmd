"안전관리" 전담 교재 의존 구조

2

반도체개발 영역의 안전 공백

30권 중 16권(53%)이 안전보건 비율 0%. 사고 사례 0건. 클린룸 출입, 시제품 테스트, VDT 증후군 등의 위험이 교재에서 전혀 다루어지지 않는다.

3

등급 1의 형식적 성격

전체 1,303건의 기본 언급 중 대부분이 "안전에 유의한다", "해당 사항 없음" 수준의 1~2줄 형식적 문구로, 실질적 안전교육 효과가 미미하다.

4

사고 사례의 절대적 부족

전체 3,552건 중 사고/부상 사례는 54건(1.5%)에 불과. 체험적 안전교육의 핵심인 사례 기반 학습이 거의 전무하다.
