# Plan: 안전보건 내용 등급 분류

## Executive Summary

| 항목 | 내용 |
|------|------|
| Feature | safety-grading (안전보건 등급 컬럼 추가) |
| 작성일 | 2026-04-02 |
| 대상 파일 | `ncs_keywords&reworkinglist_with_fullpage.xlsx` |

| 관점 | 설명 |
|------|------|
| **Problem** | 3,552행의 NCS 교재 페이지 내용에 안전보건 관련성 수준이 혼재되어 있어, 연구 분석 시 수작업 분류 필요 |
| **Solution** | E열(페이지전체내용)을 키워드 기반으로 자동 분석하여 F열에 1~3등급 부여 |
| **Function UX Effect** | Excel에서 등급 필터링으로 즉시 안전보건 밀접 페이지(등급1-2)와 무관 페이지(등급3) 분리 가능 |
| **Core Value** | 3,552행 수작업 분류를 자동화하여 연구 분석 시간 대폭 단축 |

## 등급 기준

| 등급 | 기준 | 설명 |
|:----:|------|------|
| **1** | 기본 언급 | 사고/질병/질식/중독/화학물질 등 기본 성질만 언급, "주의해야 함" 수준 |
| **2** | 구체적 대책 | 사고/질병 위험 피하기 위한 안전조치/대책, 화학물질 안전관리 방법 등 구체적 내용 제시 (1/2 페이지 이상) |
| **3** | 관련 없음 | 안전보건 주요 용어 없음 / 관련 내용 없거나 거의 없음 (5건 이하) |

## 분류 알고리즘

```
INPUT: E열(페이지전체내용)

Step 1: 안전보건 키워드 카운트
  키워드: 사고, 질병, 질식, 중독, 화학물질, 안전, 보건, 위험, 유해,
         보호구, 화재, 폭발, 감전, 추락, 끼임, 누출, MSDS, 물질안전보건자료,
         소음, 진동, 방사선, 먼지, 분진, 산업안전보건법, 작업환경,
         유의사항, 유의 사항, 주의
  → safety_count

Step 2: 조치/대책 키워드 카운트
  키워드: 안전조치, 안전대책, 예방, 방지, 관리방법, 보호장비, 안전장치,
         안전수칙, 대처, 응급조치, 환기, 차단, 격리, 착용, 조치사항,
         안전관리, 안전교육, 보호복, 안전모, 안전화, 보안경, 마스크,
         방독면, 소화기, 대피
  → measure_count

Step 3: 분류
  IF safety_count <= 5:
    → 등급 3 (관련 없음)
  ELSE IF measure_count >= 5 AND E열 길이에서 안전 관련 문장이 상당량:
    → 등급 2 (구체적 대책)
  ELSE:
    → 등급 1 (기본 언급)
```

## 구현 방식

- Python (openpyxl) 스크립트로 처리
- F열에 "등급" 헤더 + 1/2/3 값 기록
- 기존 `_with_fullpage.xlsx` 파일에 직접 추가 저장
