# 프로젝트 개요

이 문서는 AI 기술을 활용한 자동화 시스템에 대해 설명합니다.

인공지능은 현대 기술의 핵심입니다. 특히 머신러닝과 딥러닝 기술이 빠르게 발전하고 있습니다.

---

## 기술 스택

| 기술 | 설명 | 버전 |
|------|------|------|
| Python | AI 개발 언어 | 3.11 |
| TensorFlow | 딥러닝 프레임워크 | 2.15 |
| React | 프론트엔드 | 18.2 |

Python은 AI 개발에 가장 많이 사용되는 언어입니다.
