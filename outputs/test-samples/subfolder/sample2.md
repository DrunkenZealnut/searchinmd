# 데이터 분석 가이드

데이터 분석은 비즈니스 의사결정에 중요합니다.

Python을 사용한 데이터 분석이 인기를 끌고 있습니다.

---

## 분석 도구

| 도구 | 용도 | Python 라이브러리 |
|------|------|-------------------|
| Pandas | 데이터 처리 | pandas |
| NumPy | 수치 연산 | numpy |
| Matplotlib | 시각화 | matplotlib |
